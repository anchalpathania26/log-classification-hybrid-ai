# 🚀 Hybrid Log Classification System

An intelligent log classification system that combines **Regex, Machine Learning, and LLMs** to handle logs of varying complexity with high accuracy and flexibility.

---

## 📌 Overview

Logs generated by real-world systems are often inconsistent and noisy. A single approach cannot effectively classify all types of logs.

This project introduces a **hybrid framework** that dynamically selects the best classification method:

* **Regex** → for simple and structured logs
* **ML (Sentence Transformer + Logistic Regression)** → for complex patterns with labeled data
* **LLM** → for ambiguous or low-data scenarios

---

## 🧠 Key Features

* Hybrid architecture (Rule-based + ML + LLM)
* Semantic understanding using embeddings
* Intelligent fallback using LLMs
* FastAPI-based API for real-time inference
* Scalable and modular design

---

## 🏗️ Architecture

Log Input → Regex / ML Model / LLM → Final Classification
